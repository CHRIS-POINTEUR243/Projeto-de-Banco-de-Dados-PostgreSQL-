# Goodreads Database Implementation (PostgreSQL)

## Description
This project consists of a PostgreSQL database implementation inspired by the Goodreads platform, designed to manage books, users, reviews, and ratings.

## Technologies
- PostgreSQL
- SQL

## Features
- Database schema design for books, users, and reviews  
- Relationships between entities (users, books, ratings)  
- Data insertion, queries, and basic data manipulation  

## Concepts Applied
- Relational database modeling  
- Primary and foreign keys  
- SQL queries (SELECT, INSERT, JOIN)  

## How to run
Execute the SQL scripts in a PostgreSQL environment to create and populate the database.
